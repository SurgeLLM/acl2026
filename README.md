# SURGeLLM Workshop Website

This repository hosts the **SURGeLLM** workshop website for **ACL 2026**, built with **Jekyll** and deployed via **GitHub Pages**.

The live site is under `acl2026/` (the root path redirects to the latest edition).

## Project Structure

- Workshop pages (ACL 2026): `acl2026/`
- Content data (edit here first): `_data/acl2026/*.yml`
  - `site.yml`, `navigation.yml`, `dates.yml`, `submission.yml`
  - `speakers.yml`, `organizers.yml`, `committee.yml`, `topics.yml`, `program.yml`, `faq.yml`, `updates.yml`
- Templates: `_layouts/` and `_includes/` (sections in `_includes/sections/`)
- Assets: `assets/css/`, `assets/js/`, images in `assets/`
- Search index: `search.json` (generated by Jekyll; used by the search modal)

## Local Development

Prereqs: Ruby + Bundler.

```bash
bundle install
bundle exec jekyll serve --config _config.yml,_config.dev.yml
```
